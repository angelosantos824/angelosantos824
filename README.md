# 🏎️ DOZE CAR - Stand de Automóveis (Vagos, PT)

Website premium desenvolvido para um stand de automóveis usados, focado em experiência do utilizador e design "Dark Mode".

## 🚀 Funcionalidades
- **Estoque Dinâmico:** Sistema de vitrine com aviso de renovação de stock.
- **Área do Cliente:** Painel personalizado para consulta de extratos e pagamentos (ex: MBWay).
- **Portal de Notícias:** Balões interativos (Modals) com dicas de manutenção e mercado.
- **Campanhas:** Secção de promoções com gatilhos de urgência.
- **Google Maps Integrado:** Localização personalizada em Vagos com filtro dark.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Flexbox & Grid)
- JavaScript (Manipulação de DOM e Modals)
- Google Maps API (Embed)

  # 🍕 Bella Pizza - Website Artesanal

O website oficial da **Bella Pizza**, criado para transmitir a essência da verdadeira pizza italiana: massa artesanal, fermentação lenta e o calor do forno a lenha. O design foca-se na experiência visual e no conforto do cliente.

## ✨ Características Principais
- **Hero Section Impactante:** Imagens de alta qualidade com gradiente para destaque do título e apelo visual.
- **Cardápio em Grid:** Organização intuitiva dos pratos favoritos com preços em destaque.
- **Botão de Delivery:** Chamada de ação estratégica para pedidos online.
- **Design Responsivo:** Adaptado para telemóveis e computadores, mantendo a legibilidade e a elegância.
- **Tipografia Premium:** Uso de fontes *Playfair Display* e *Poppins* para um toque clássico e moderno.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Estilização com Flexbox e Grid)
- **Google Fonts** (Playfair Display & Poppins)
- **Unsplash API** (Imagens de alta resolução)

## 🚀 Como Executar

2. Abre a pasta do projeto.
3. Abre o ficheiro `index.html` no teu navegador preferido.
